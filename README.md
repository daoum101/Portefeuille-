# ViteAuto 100/100 - voiture exacte validée

## Inclus
- voiture exacte validée en hero
- bouton Rechercher moins rouge / plus propre
- frontend React + Vite
- backend FastAPI
- structure prête pour GitHub

## Frontend
```bash
cd frontend
npm install
npm run dev
```

## Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload --port 8000
```
